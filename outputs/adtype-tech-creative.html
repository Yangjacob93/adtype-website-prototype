PROBLEM

왜 필요한가요

전략이 좋아도 제작 과정에서 개인의 취향과 감이 앞서면 의도가 흐려집니다. OOH는 보는 시간이 짧고, 거리와 규격이 다르기 때문에 전략을 공간에 맞게 번역해야 합니다.

SYSTEM DEFINITION

Key Visual / Creative이란

데이터 기반 캠페인 전략을 OOH 환경에 맞는 비주얼 컨셉, 카피, 캐릭터, 공간, 스토리보드, 소재 베리에이션으로 변환하는 시스템입니다.
