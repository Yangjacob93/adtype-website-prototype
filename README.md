# Adtype Website Prototype

애드타입 홈페이지 및 상세 페이지 제작 작업물입니다.

## 주요 파일

- `outputs/adtype-for-brands.html` - 브랜드사 대상 메인 페이지
- `outputs/adtype-brand-presence.html` - 브랜드 존재감 확대 상세 페이지
- `outputs/adtype-brand-positioning.html` - 브랜드 포지셔닝 강화 상세 페이지
- `outputs/adtype-brand-purchase-reason.html` - 구매 이유 형성 상세 페이지
- `outputs/adtype-brand-launch-renewal.html` - 신제품/리뉴얼 인식 전환 상세 페이지
- `outputs/adtype-brand-offline-traffic.html` - 오프라인 접점 기반 유입 확대 상세 페이지
- `outputs/adtype-brand-performance-loop.html` - 브랜드 지표 개선과 캠페인 고도화 상세 페이지
- `outputs/assets/` - 영상, 이미지, 로고, PDF 에셋
- `work/` - 빌드/QA용 보조 스크립트와 참고 텍스트

## 제외한 파일

GitHub 업로드 용량을 줄이기 위해 재생성 가능한 아임웹 패키지 ZIP, 중복 패키지 폴더, QA 캡처 이미지는 `.gitignore`로 제외했습니다.
